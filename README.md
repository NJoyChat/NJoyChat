Schritt 1:
Nach der Installation von Tampermonkey (https://www.tampermonkey.net/) sollte es in der Addon Leiste erscheinen.
Bei einem Linksclick öffnet sich ein Menü. Hier gibt es unter anderem die Option ein neues Userscript zu erstellen.
Diese wählen.
![Step1](https://raw.githubusercontent.com/NJoyChat/NJoyChat/master/step1.png)
Es öffnet sich dieses Fenster:
![Step2](https://raw.githubusercontent.com/NJoyChat/NJoyChat/master/step2.png)
Diesen Link
https://raw.githubusercontent.com/NJoyChat/NJoyChat/master/NJoyChat.js
in einem neuen Tab öffnen, komplett markieren (STRG + A) und kopieren (STRG + C) 
![Step3](https://raw.githubusercontent.com/NJoyChat/NJoyChat/master/step3.png)
Hier erneut STRG + A drücken um alles zu markieren und danach STRG + V um den vorher kopierten Text einzufügen.
danach auf speichern drücken.
![Step4](https://raw.githubusercontent.com/NJoyChat/NJoyChat/master/step4.png)
Oben im Reiter "Einstellungen" kontrollieren ob die Update Einstellungen korrekt eingestellt sind. Der update link ist der link von oben.
